Misc

기타 잡다한 문제로 구성되며 기본적인 CTF 툴 활용법이나 트릭이 나옵니다.

풀이 방법

  • 문제 분석 (파일 형식 확인, 숨겨진 정보 찾기 등)
  • 기초 명령어 및 유틸리티 사용

추천 툴/사이트

  • magic-number
    파일 포맷 식별 도구
  • strings
    텍스트 추출 유틸리티

Pwn

버퍼 오버플로우, 포맷 스트링 공격 등 바이너리 공격 중심 문제입니다.

풀이 방법

  • gdb, pwndbg를 활용한 디버깅
  • ROP, shellcode 삽입

추천 툴/사이트

  • gdb-pwndbg
    Pwn용 gdb 확장
  • Dreamhack Labs
    CTF 연습 플랫폼

Web

XSS, SQL Injection, 파일 업로드 등 웹 취약점을 이용하는 문제입니다.

풀이 방법

  • 브라우저 개발자 도구 사용
  • BurpSuite를 이용한 요청 조작

추천 툴/사이트

  • Burp Suite
    웹 요청 분석 및 조작 도구
  • PortSwigger
    웹 해킹 학습 플랫폼

Reversing

역공학 문제로 바이너리 실행 파일 분석을 통해 로직을 파악하는 문제입니다.

풀이 방법

  • IDA, Ghidra 등 디스어셈블러 사용
  • gdb를 통한 동적 분석

추천 툴/사이트

  • Ghidra
    NSA에서 개발한 무료 리버싱 도구
  • IDA Free
    유명한 상용 리버싱 도구의 무료 버전

Network/Pentest

네트워크 패킷 캡처 분석, 포트 스캐닝 등 침투 테스트 기반 문제입니다.

풀이 방법

  • Wireshark로 패킷 분석
  • nmap으로 포트/서비스 스캔

추천 툴/사이트

  • Wireshark
    패킷 분석 도구
  • nmap
    네트워크 스캐닝 도구

Crypto

고전 암호, 대칭/비대칭키, 해시 등을 이용한 암호학 문제입니다.

풀이 방법

  • 암호 알고리즘 종류와 원리 학습
  • 수학적 추론을 통한 키나 평문 추출

추천 툴/사이트

  • CyberChef
    암호 해독 및 변환 도구
  • Cryptopals
    암호학 학습 문제 모음

Forensics

디지털 포렌식 문제로 메모리, 파일 시스템, 이미지 분석 등을 포함합니다.

풀이 방법

  • Autopsy 등으로 디스크 이미지 분석
  • ExifTool로 메타데이터 확인

추천 툴/사이트

  • Autopsy
    포렌식 이미지 분석 도구
  • ExifTool
    메타데이터 추출 도구

Cooldown

보통 마지막에 쉬어가는 문제로, 유머나 창의력을 요하는 문제입니다.

풀이 방법

  • 직관적 사고, 창의적 접근
  • 파일 숨김, 암시된 트릭 등을 탐색

추천 툴/사이트

  • HackTheBox
    다양한 장르 문제 수록된 플랫폼
  • OverTheWire
    기초부터 실습 가능한 게임형 CTF
