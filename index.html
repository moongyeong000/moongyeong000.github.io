Pwn

바이너리 익스플로잇을 통해 시스템 권한을 얻는 문제 유형입니다.

  • GDB + Pwndbg: 강력한 GDB 플러그인으로, 메모리 분석과 ROP 디버깅에 사용
  • Ghidra: NSA가 개발한 무료 디컴파일러, 정적 분석에 유용
  • IDA: 산업 표준의 리버싱 도구, 바이너리 구조 분석에 사용

Web

웹 애플리케이션의 취약점을 이용하는 문제입니다.

  • Burp Suite: 웹 요청 가로채기 및 리플레이 기능을 제공
  • strings: 바이너리 내 문자열 검색 도구로 숨겨진 URL 또는 정보 탐색

Reversing

바이너리를 분석하여 숨겨진 로직이나 플래그를 찾는 문제입니다.

  • radare2: 커맨드 기반의 강력한 리버싱 툴
  • Ghidra: GUI를 통한 디컴파일, 흐름 제어 및 API 추적에 강점
  • IDA: 디스어셈블 결과를 시각적으로 확인 가능

Forensics

디지털 흔적 분석을 통해 데이터를 복구하거나 정보를 추출하는 문제입니다.

  • Wireshark: 패킷 분석 도구, 네트워크 트래픽 감시
  • Autopsy: 디스크 이미지 분석 및 파일 복구
  • exiftool: 메타데이터 분석을 위한 도구

Misc

기타 암호, 트리비아, OSINT 등 다양한 유형의 문제입니다.

  • nmap: 네트워크 포트 스캐너, 서비스 탐지에 사용
  • Magic Number Finder: 파일 헤더 식별을 통한 파일 유형 분석
