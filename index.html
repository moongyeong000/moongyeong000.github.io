Misc

기타 문제들을 포함하는 카테고리로, 파일 포맷 분석, 숨겨진 정보 추출 등 다양한 기초적인 분석 기술을 요구합니다.

풀이 방법

  • 파일 형식 식별
  • 문자열 추출
  • 파일 내부 숨겨진 정보 확인

추천 툴/사이트

  • magic-number
    파일 포맷을 식별하는 데 사용하는 도구
  • strings
    바이너리 파일 내 문자열 추출 유틸리티

Pwn

버퍼 오버플로우, 포맷 스트링 등 바이너리 취약점을 이용한 공격 문제입니다.

풀이 방법

  • gdb 및 pwndbg로 디버깅
  • BOF, ROP, shellcode 등 기법 사용

추천 툴/사이트

  • gdb-pwndbg
    디버깅과 취약점 분석용 GDB 확장 도구
  • radare2
    강력한 오픈소스 역공학 프레임워크
  • Dreamhack Labs
    CTF 문제를 실습해볼 수 있는 한국 플랫폼

Web

XSS, SQL Injection, 파일 업로드 취약점 등 웹 해킹 기법을 이용한 문제입니다.

풀이 방법

  • 개발자 도구로 웹 요청 및 DOM 구조 분석
  • BurpSuite 등을 이용한 요청/응답 조작

추천 툴/사이트

  • Burp Suite
    웹 요청 조작 및 분석 툴
  • PortSwigger
    BurpSuite 제작사의 학습 자료 제공

Reversing

바이너리를 디스어셈블하거나 디컴파일하여 내부 동작을 분석하는 문제입니다.

풀이 방법

  • Ghidra, IDA 등으로 정적 분석
  • gdb 또는 radare2로 동적 분석

추천 툴/사이트

  • Ghidra
    NSA가 개발한 무료 역공학 도구
  • IDA Free
    전통적인 상용 리버싱 도구의 무료 버전

Network/Pentest

네트워크 트래픽 분석, 포트 스캔, 서비스 정보 수집 등 침투 테스트 관련 문제입니다.

풀이 방법

  • Wireshark로 패킷 캡처 분석
  • nmap으로 포트 및 서비스 탐지

추천 툴/사이트

  • Wireshark
    패킷 캡처 및 분석 도구
  • nmap
    네트워크 스캐너

Crypto

암호학 기반 문제로, 암호 알고리즘과 관련된 분석과 수학적 사고를 필요로 합니다.

풀이 방법

  • 고전 암호 및 현대 암호 알고리즘 학습
  • 암호 해독, 수학적 추론

추천 툴/사이트

  • CyberChef
    데이터 인코딩, 해싱, 암호화 등을 쉽게 수행
  • Cryptopals
    암호학 기초 학습 문제 모음

Forensics

디스크 이미지, 메모리 덤프, 사진 등의 증거 파일을 분석하는 문제입니다.

풀이 방법

  • 디스크 분석 도구로 파일 복구, 시간 분석
  • ExifTool을 통한 메타데이터 수집

추천 툴/사이트

  • Autopsy
    디지털 포렌식 분석 플랫폼
  • ExifTool
    이미지 및 파일 메타데이터 추출 도구

Cooldown

창의력과 직관을 요구하는 쉬어가는 문제들입니다. 유머, 눈치, 트릭성 문제들이 포함됩니다.

풀이 방법

  • 직관적인 접근
  • 문제명, 설명, 힌트에서 유추

추천 툴/사이트

  • HackTheBox
    실전 기반 해킹 실습 플랫폼
  • OverTheWire
    초보자를 위한 단계별 해킹 게임
