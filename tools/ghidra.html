Ghidra

NSA에서 개발한 강력한 리버스 엔지니어링 도구로, 바이너리 분석 및 디스어셈블리/디컴파일링에 사용됩니다.

🔗 다운로드 링크

  • 공식 Ghidra 다운로드 페이지

⚙️ 설치 및 사용법

# Java 설치 필요
$ unzip ghidra_*.zip
$ cd ghidra_*
$ ./ghidraRun

# 바이너리 파일을 프로젝트에 추가 후 분석 시작

코드를 자동으로 디컴파일하여 고수준 언어(C와 유사한 형태)로 보여주는 기능이 핵심입니다.

🧩 사용된 문제 예시

  • picoCTF - "Safe Opener", "asm1", "Vault Door" 시리즈
  • Google CTF - 리버스 엔지니어링 문제
  • DEFCON Quals - 다양한 바이너리 문제
