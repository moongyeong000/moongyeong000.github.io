Autopsy

디지털 포렌식 분석 툴로, 하드 드라이브와 스마트폰에서 증거를 수집, 분석, 보고하는 데 사용됩니다.

🔗 다운로드 링크

  • Autopsy 공식 다운로드

⚙️ 설치 및 사용법

# 설치 (Ubuntu 예시)
sudo apt install autopsy

# Autopsy 실행
autopsy

# 웹 인터페이스 접속 후 증거 수집 및 분석 수행

그래픽 웹 기반 인터페이스를 통해 손쉽게 파일 시스템 및 데이터 복구 작업을 할 수 있습니다.

🧩 사용된 문제 예시

  • DFIR CTF - 디지털 포렌식 증거 분석
  • CyberDefenders - 삭제된 파일 복구 및 타임라인 분석
  • SANS Forensics Challenge - 메모리 및 파일 시스템 분석

📂 관련 카테고리

  • Forensics
  • Disk Analysis
  • Memory Forensics
