Magic Number

파일의 헤더(Signature)를 분석하여 파일 형식을 식별하는 도구입니다.

🔗 다운로드 링크

  • 파일 시그니처 목록 (Wikipedia)
  • file 명령어 매뉴얼

⚙️ 사용법

$ file suspicious_file  
suspicious_file: PNG image data, 600 x 400, 8-bit/color RGBA, non-interlaced

파일 확장자와 무관하게 파일 내용을 기반으로 형식을 확인할 수 있습니다.

  • Wargame Bandit - Level 12
  • CTFlearn - "File Type Investigation"
