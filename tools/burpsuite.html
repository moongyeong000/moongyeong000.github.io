Burp Suite

웹 취약점 진단 및 트래픽 가로채기 도구로, CTF의 웹 문제에서 거의 필수적인 툴입니다.

🔗 다운로드 링크

  • 공식 사이트 (Community Edition 무료 제공)

⚙️ 설치 및 사용법

# Linux에서 실행 예시 (java 필요)
$ java -jar burpsuite_community.jar

# 브라우저 프록시 설정 (기본 포트: 127.0.0.1:8080)
# Burp Proxy 탭에서 Intercept 사용 설정

Burp는 웹 애플리케이션과 클라이언트 사이의 요청/응답을 분석하고 조작할 수 있게 해줍니다.

🧩 사용된 문제 예시

  • picoCTF - "Insp3ct0r", "Where are the robots?"
  • HTB - 여러 웹 문제들
  • Korea CTF - "Web Upload", "Simple Auth"
