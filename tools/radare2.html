Radare2

강력한 오픈소스 역공학 프레임워크로, 디버깅, 디스어셈블링, 바이너리 분석 등 다양한 기능을 제공합니다.

🔗 다운로드 링크

  • 공식 사이트: https://rada.re/n/
  • GitHub 저장소

⚙️ 설치 및 사용법

# 설치
$ git clone https://github.com/radareorg/radare2.git
$ cd radare2
$ ./sys/install.sh

# 실행 예시
$ r2 binary
[0x00000000]> aaa         ; 분석 자동 실행
[0x00000000]> afl         ; 함수 목록 보기
[0x00000000]> pdf @ main  ; main 함수 디스어셈블
[0x00000000]> V           ; 그래픽 모드

Radare2는 CLI 기반으로 작동하며 다양한 명령어가 존재합니다.
