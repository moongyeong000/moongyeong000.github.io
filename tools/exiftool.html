ExifTool

이미지, 비디오, 오디오 등 다양한 파일의 메타데이터를 읽고 수정할 수 있는 강력한 커맨드라인 도구입니다.

🔗 다운로드 링크

  • ExifTool 공식 사이트

⚙️ 설치 및 사용법

# 설치 (Ubuntu 예시)
sudo apt install libimage-exiftool-perl

# 메타데이터 읽기
exiftool sample.jpg

# 메타데이터 수정 (예: 작성자 변경)
exiftool -Artist="Your Name" sample.jpg

ExifTool은 거의 모든 파일 포맷의 메타데이터를 지원하며, 디지털 포렌식 및 CTF에서 흔히 쓰입니다.

🧩 사용된 문제 예시

  • Forensics CTF - 숨겨진 메타데이터 정보 발견
  • Image Analysis Challenge - EXIF 정보 기반 타임라인 분석
  • Steganography Challenge - 파일 메타데이터 변조 및 탐지
