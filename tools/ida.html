IDA Pro

Hex-Rays 사에서 개발한 상용 리버스 엔지니어링 툴로, 정적 분석과 디컴파일 기능에서 최고의 성능을 자랑합니다.

🔗 다운로드 링크

  • 공식 IDA Free 버전 다운로드

⚙️ 설치 및 사용법

# Windows 설치:
- 설치 후 IDA 실행
- 바이너리 로딩 > 분석 시작

# Linux 실행 (Free 버전은 Wine 필요할 수 있음):
$ wine idaq.exe

기본 분석을 수행하면 함수 트리, 문자열, 연관된 함수 호출 흐름을 시각적으로 확인할 수 있습니다.

🧩 사용된 문제 예시

  • picoCTF - "KeygenMe" 계열
  • Real World CTF - 고급 리버싱 문제
  • CTFtime 상위 대회 대부분의 Reverse 문제
