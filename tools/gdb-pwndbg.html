gdb-pwndbg

디버깅 및 익스플로잇 개발을 도와주는 GDB 확장 도구

다운로드 / 설치

GitHub에서 다운로드

git clone https://github.com/pwndbg/pwndbg
cd pwndbg
./setup.sh

사용법 예시

다음은 pwndbg를 이용해 프로그램을 디버깅하는 예시입니다.

gdb ./vulnerable_binary
# pwndbg 설치 후 다양한 UI 및 분석 기능 제공

사용된 CTF 문제 예시

  • Dreamhack - basic_exploitation_001
  • picoCTF - buffer overflow 1~3

관련 카테고리

  • Pwn
  • Reversing (보조 분석용)

CTF 학습용 미니 퀴즈

문제: pwndbg는 GDB의 기본 UI를 완전히 대체한다. (O/X)

정답 보기

✗. pwndbg는 GDB 위에 다양한 기능을 추가하지만 기본 GDB 명령도 그대로 사용합니다.

실전 체크리스트

  • GDB가 올바르게 설치되었는지 확인
  • pwndbg의 Python 요구사항 충족 (Python 3.8 이상 권장)
  • 디버깅할 바이너리에 심볼 정보가 포함되어 있는지 확인
  • ASLR, NX 등 보안 설정 확인
