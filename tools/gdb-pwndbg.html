gdb-pwndbg

기능 개요

gdb-pwndbg는 GDB를 위한 확장 플러그인으로, 바이너리 익스플로잇 분석을 보다 직관적이고 강력하게 만들어 줍니다. 레지스터, 스택, 힙 등의 상태를 실시간으로 보기 좋게 출력해 줍니다.

사용 방법

  • gdb에 pwndbg를 설치: git clone https://github.com/pwndbg/pwndbg 후 ./setup.sh
  • 익스플로잇 대상 바이너리를 gdb로 실행: gdb ./vulnerable
  • 디버깅 중 스택 프레임, 레지스터, 메모리 등을 시각적으로 확인 가능

활용 예시

  • 스택 버퍼 오버플로우 위치 및 크기 파악
  • RET 주소 덮어쓰기 위치 분석
  • 힙 구조 분석 및 double free, UAF 등 취약점 식별
