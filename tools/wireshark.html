Wireshark

네트워크 패킷을 캡처하고 분석하는 강력한 오픈소스 툴입니다. 다양한 프로토콜을 지원하여 트래픽 분석 및 네트워크 문제 해결에 필수적입니다.

  • 공식 Wireshark 다운로드

⚙️ 설치 및 사용법

# Windows, macOS, Linux 모두 지원
- 설치 후 네트워크 인터페이스 선택
- 실시간 패킷 캡처 또는 저장된 pcap 파일 열기
- 필터 사용: ip.addr == 192.168.0.1, tcp.port == 80 등
- 패킷 상세 분석 및 프로토콜 디코딩

필터링과 프로토콜 해석을 통해 네트워크 공격 흔적, 데이터 탈취 시도 등을 탐지할 수 있습니다.

🧩 사용된 문제 예시

  • DEF CON CTF - 네트워크 트래픽 분석 문제
  • picoCTF - 패킷 캡처 기반 문제
  • Google CTF - HTTP, DNS 프로토콜 취약점 분석

📂 관련 카테고리

  • Network Forensics
  • Packet Analysis
  • Traffic Inspection
