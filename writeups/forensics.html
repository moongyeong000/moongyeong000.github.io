Forensics Writeups

포렌식 문제 관련 이미지 분석, 파일 복구, 메모리 덤프, 타임스탬프 분석 등 문제 풀이 모음입니다.

추천 문제

  • Memory Dump Analysis - CTFTime 문제 링크
    메모리 덤프에서 중요한 정보를 추출하는 문제입니다.
  • Image Steganography - CTFTime 문제 링크
    이미지 내 숨겨진 데이터를 찾아내는 기법을 연습할 수 있습니다.
  • Timestamp Forensics - CTFTime 문제 링크
    파일의 타임스탬프를 분석하여 힌트를 얻는 문제입니다.

주요 사용 도구

  • Autopsy - 디지털 포렌식 분석 툴
  • ExifTool - 파일 메타데이터 추출 및 분석 도구
  • binwalk - 파일 내 숨겨진 데이터 추출 도구

학습 팁

  • 파일 포맷과 구조를 이해하는 것이 매우 중요합니다.
  • 메타데이터와 타임스탬프에서 유용한 정보를 찾아내는 법을 연습하세요.
  • 숨겨진 데이터를 찾기 위한 다양한 도구 활용법을 익히세요.
