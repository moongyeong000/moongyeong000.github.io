[Steganography] Hidden Cat

문제 설명: 주어진 이미지 파일에 숨겨진 메시지를 찾는 문제입니다.

풀이 과정:

  1. file cat.png으로 PNG 포맷 확인
  2. zsteg cat.png 실행 → LSB에서 base64 인코딩 문자열 발견
  3. base64 디코딩 후 flag{hidden_in_pixels} 획득

[Encoding] Morse Flag

문제 설명: 모스 부호로 인코딩된 문자열이 주어졌습니다.

풀이 과정:

  1. 문자열을 CyberChef의 Morse Code Decode 연산자로 디코딩
  2. 정상적인 영어 문장 획득 후 flag{morse_decoded_text} 추출
