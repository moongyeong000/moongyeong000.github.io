[RSA] Easy RSA

문제 설명: 공개키 (n, e) 와 암호문 c 가 주어짐. n이 작은 소수의 곱인 경우.

풀이 과정:

  1. Factordb에 n 입력 → p, q 분해됨
  2. 파이 함수 φ(n) 계산: (p-1)*(q-1)
  3. 개인키 d 계산: d = pow(e, -1, φ(n))
  4. 복호화: m = pow(c, d, n) → 평문 획득 후 flag{...} 포맷 확인

[Classic Cipher] Caesar Attack

문제 설명: 시저 암호로 인코딩된 문자열에서 원문을 복원

풀이 과정:

  1. CyberChef에서 Caesar Decode 사용
  2. Shift 수치를 조정하여 올바른 평문 도출
  3. flag 형식에 맞는 문자열 flag{shift_success} 획득
