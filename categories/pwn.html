🗺️ Pwn 문제 풀이 흐름도

🔎 Pwn 문제 유형 & 대표 예시

  • Buffer Overflow: 스택 버퍼 오버플로우, 리턴 주소 덮어쓰기
  • Format String: 포맷 스트링 취약점, 임의 메모리 읽기/쓰기
  • ROP/Ret2libc: Return Oriented Programming, 라이브러리 함수 호출
  • Heap Exploitation: 힙 오버플로우, Use-After-Free, Double Free 등

🛠️ Pwn 문제 추천 도구/사이트

  • GDB / pwndbg: 바이너리 디버깅, 메모리 분석
    실행 중인 프로그램의 메모리, 레지스터, 스택 추적
  • Ghidra: 정적 분석, 함수/변수 구조 파악
    오픈소스 리버스 엔지니어링 도구
  • pwntools: 익스플로잇 자동화 파이썬 라이브러리
    익스플로잇 작성, 원격 서버와 통신, 페이로드 생성

💡 실전 체크리스트 & 주의사항

  • NX, PIE, ASLR 등 보안 옵션 확인
  • 입력값 길이, 오프셋, 페이로드 구조 파악
  • 디버깅/분석 도구 적극 활용
  • 실제 문제 예시/Writeup: Pwn Writeups 바로가기
