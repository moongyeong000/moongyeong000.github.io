주요 문제 유형 & 공략법

  • 인코딩/디코딩

    정의: Base64, Hex, URL 등 다양한 인코딩/디코딩을 활용한 문제
    공격 가능한 조건:
    • 문제에서 인코딩된 문자열, 파일 등 제공
    공격 방법:
    • CyberChef, Python 등으로 인코딩/디코딩 반복

  • 파일 포맷 분석

    정의: 파일 헤더, 마법값, 구조를 분석해 숨겨진 데이터나 플래그를 찾는 문제
    공격 가능한 조건:
    • 의심스러운 파일, 확장자와 내용 불일치 등
    공격 방법:
    • binwalk, file, hex editor 등으로 파일 구조 분석

  • 자동화 스크립트

    정의: 반복 작업, 대량 데이터 처리, 변환 등을 자동화하는 문제
    공격 가능한 조건:
    • 규칙성 있는 데이터, 반복 변환 필요
    공격 방법:
    • Python, Bash 등으로 스크립트 작성

주요 도구 & 툴

  • CyberChef

    설명: 다양한 인코딩, 디코딩, 변환, 암호화/복호화 기능을 제공하는 온라인 도구
    주요 기능:
    • Base64, URL, Hex, ROT13, XOR 등 다양한 변환
    • 여러 변환을 조합하여 한 번에 처리 가능
    활용 예시:
    • 문제에서 받은 문자열, 파일을 빠르게 변환/복호화

  • file / exiftool

    설명: 파일의 타입, 메타데이터, 숨겨진 정보 등을 분석하는 도구
    주요 기능:
    • 파일 헤더, 메타데이터, 숨겨진 데이터 추출
    활용 예시:
    • 파일 포맷 판별, 이미지/문서 내 숨겨진 정보 확인

  • Python/Bash

    설명: 데이터 변환, 반복 작업, 자동화 스크립트 작성에 활용
    주요 기능:
    • 텍스트/파일 처리, 인코딩/디코딩, 대량 데이터 변환
    활용 예시:
    • 규칙성 있는 데이터 변환, 반복 작업 자동화
