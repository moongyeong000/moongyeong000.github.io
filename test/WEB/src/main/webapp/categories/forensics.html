주요 문제 유형 & 공략법

  • 디스크/이미지 분석

    정의: 디스크, USB, 메모리 덤프 등에서 플래그나 중요한 정보를 추출하는 문제
    공격 가능한 조건:
    • 디스크/이미지 파일, 메모리 덤프 등 제공
    공격 방법:
    • 파일 시스템 분석, 삭제 파일 복구, 숨겨진 데이터 추출

  • 네트워크 패킷 분석

    정의: 캡처된 패킷(pcap)에서 플래그, 파일, 인증 정보 등을 찾는 문제
    공격 가능한 조건:
    • Wireshark 등으로 열 수 있는 pcap 파일 제공
    공격 방법:
    • 프로토콜 분석, 파일 추출, 평문 데이터/쿠키/세션 분석

  • Steganography (스테가노그래피)

    정의: 이미지, 오디오, 영상 파일 등에 숨겨진 데이터를 찾아내는 문제
    공격 가능한 조건:
    • 의심스러운 미디어 파일 제공
    공격 방법:
    • LSB, 메타데이터, 히든 섹션, 파일 포맷 분석

주요 도구 & 툴

  • Autopsy / FTK Imager

    설명: 디스크/이미지 파일의 파일 시스템, 삭제 파일, 메타데이터 분석 도구
    주요 기능:
    • 파일 복구, 타임라인 분석, 해시 비교
    활용 예시:
    • 삭제된 플래그 파일 복구, 파일 생성/수정 이력 추적

  • Wireshark

    설명: 네트워크 패킷(pcap) 분석 및 프로토콜 디코딩 도구
    주요 기능:
    • TCP/UDP/HTTP 등 다양한 프로토콜 분석
    • 파일 추출, 필터링, 스트림 재조립
    활용 예시:
    • 패킷 내 평문 플래그, 파일, 인증 정보 추출

  • StegSolve / StegOnline

    설명: 이미지 파일의 LSB, 컬러 채널, 메타데이터 등 숨겨진 정보를 시각적으로 분석하는 도구
    주요 기능:
    • 채널 분리, LSB 추출, 이미지 변환
    활용 예시:
    • 이미지에 숨겨진 플래그, 텍스트, 파일 추출

  • binwalk

    설명: 바이너리 파일에서 임베디드 데이터, 파일 시스템, 압축 파일 등을 자동으로 추출하는 도구
    주요 기능:
    • 파일 시그니처 스캔, 자동 추출, 파일 시스템 분석
    활용 예시:
    • 펌웨어, 이미지 파일에서 숨겨진 데이터 추출
