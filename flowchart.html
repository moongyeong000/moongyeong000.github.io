CTF 문제 유형별 기본 풀이 절차

1. Recon (정보 수집)

  1. 문제 설명 및 파일 확인
  2. 파일 형식, 크기, 해시 체크
  3. strings, file 명령어 사용
  4. 네트워크 문제면 포트 스캔 및 트래픽 캡처

2. 분석 (정적/동적)

  1. 바이너리면 Ghidra, IDA, radare2 등으로 정적 분석
  2. 디버깅 필요 시 gdb, pwndbg 사용
  3. 웹이면 BurpSuite, 개발자 도구 활용
  4. 암호 문제는 알고리즘 분석 및 수학적 접근

3. 익스플로잇 및 테스트

  1. 취약점 포착 후 익스플로잇 작성
  2. 페이로드 제작 및 버퍼 오버플로우, ROP 시도
  3. 웹 요청 변조 및 페이로드 주입
  4. 암호 해독 및 스크립트 작성

4. 플래그 획득 및 검증

  1. 출력된 플래그 포맷 확인
  2. 제출 전에 플래그 유효성 검사
  3. Writeup 작성 준비

5. Writeup 작성 및 정리

  1. 문제 설명과 풀이 과정 기록
  2. 사용한 도구 및 명령어 정리
  3. 재현 가능한 코드 및 스크립트 첨부
  4. 팀원 공유 및 복습
