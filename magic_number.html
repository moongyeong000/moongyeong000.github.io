magic-number

기능 개요

파일의 바이너리 시그니처(매직 넘버)를 검사하여 파일 형식을 판별하는 데 사용됩니다.

사용 방법

  • file <filename> 명령으로 간단히 확인 가능
  • 정확한 바이너리 헤더를 통해 파일 포맷 식별 가능 (예: PNG → 89 50 4E 47)
  • Hex editor로 열어 매직 넘버 직접 확인 가능

활용 예시

  • 확장자가 잘못된 파일 확인
  • 변형된 이미지, 문서 파일의 진짜 포맷 파악
  • 스테가노그래피 문제에서 파일 숨김 여부 확인
